EPL Season Performance Analyzer

Problem Definition

This program investigates a core question: Do Premier League soccer teams tend to perform better at the beginning of a season or toward the end? By analyzing match data across multiple seasons, this tool provides key statistics, performance trends, and for a bit of fun a match outcome prediction model based on team form (Elo rating).
